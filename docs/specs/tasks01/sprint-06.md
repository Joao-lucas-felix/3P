# Sprint 06

## Objetivo

Completar a trilha de conceitos e conectar cada bloco a pratica.

## Alteracao minima esperada

- adicionar os blocos de Matematica e Ad-Hoc;
- adicionar chamadas contextualizadas para o repositorio de exercicios ou resolucoes em todos os conceitos;
- incluir um fechamento incentivando a continuidade do estudo.

## Valor entregue

Esta sprint fecha a primeira versao funcional da pagina de conceitos, conectando teoria, progressao e pratica.

## Teste a criar ou ajustar antes da implementacao

- teste que valida a existencia dos nove conceitos obrigatorios;
- teste que valida que cada conceito possui uma chamada contextualizada para exercicios ou resolucoes;
- teste que valida a existencia de um fechamento de continuidade;
- teste que valida que a pagina nao exibe texto de solicitacao tecnica ou instrucao de desenvolvimento.

## Evidencia de falha esperada antes de implementar

- a pagina ainda nao esta completa;
- nem todos os conceitos possuem ponte para pratica;
- os testes finais devem falhar antes da implementacao.

## Implementacao esperada

- concluir os dois conceitos restantes;
- adicionar links contextualizados para pratica;
- reforcar a progressao e o proximo passo do estudante;
- manter o tema visual atual sem alteracoes de identidade.

## Documentacao da execucao

Ao concluir a sprint, registrar:

- como os links para pratica foram apresentados;
- como a pagina foi fechada pedagogicamente;
- quais testes finais garantem o comportamento esperado.

## Criterio de conclusao

- os nove conceitos obrigatorios estao presentes;
- todos possuem ponte para pratica;
- a pagina tem fechamento;
- os testes da sprint passam.

## Registro de execucao

- testes finais ampliados para cobrar os nove conceitos, as pontes para pratica e o fechamento da pagina;
- blocos de Matematica e Ad-Hoc adicionados para completar a trilha prevista no PRD;
- cada conceito passou a incluir uma chamada contextualizada para exercicios e resolucoes no repositorio de referencia;
- secao final de continuidade adicionada para incentivar o proximo passo do estudante;
- o tema visual atual foi preservado durante toda a conclusao da pagina.

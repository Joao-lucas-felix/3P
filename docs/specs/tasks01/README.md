# Tasks 01

Este diretorio organiza a implementacao do [PRD 01](C:\Users\jlmac\OneDrive\Documentos\dev\3P\docs\specs\PRD_01\PRD.md) em sprints pequenas, com foco em:

- uma alteracao minima por sprint;
- teste criado ou ajustado antes da implementacao;
- documentacao objetiva do que foi executado.

## Sequencia proposta

1. `sprint-01.md`: base navegavel da pagina de conceitos.
2. `sprint-02.md`: abertura pedagogica da pagina.
3. `sprint-03.md`: primeiro bloco de conceito validando o modelo.
4. `sprint-04.md`: progressao inicial com condicao, repeticao e decomposicao.
5. `sprint-05.md`: listas, matrizes e strings.
6. `sprint-06.md`: fechamento da trilha com matematica, ad-hoc e links para pratica.
7. `sprint-07.md`: reorganizacao da pagina por grupos de conceitos para reduzir a quantidade de cards.

## Regra geral

Cada sprint deve:

- revisar os testes existentes;
- criar ou ajustar testes antes da implementacao;
- confirmar a falha esperada;
- implementar o menor incremento util;
- registrar o que foi entregue.

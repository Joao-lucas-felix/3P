# Tasks 04

Este diretório organiza a implementação do [PRD 04](C:\Users\jlmac\OneDrive\Documentos\dev\3P\docs\specs\PRD_04\PRD.md) em sprints pequenas, com foco em:

- uma alteração visual ou estrutural por sprint;
- teste criado ou ajustado antes da implementação;
- mudança mínima no código a cada etapa;
- documentação objetiva do que foi executado.

## Estratégia

A `task04` foi dividida em camadas para permitir um redesign forte sem perder controle:

1. `sprint-01.md`: base visual compartilhada da nova direção.
2. `sprint-02.md`: navegação global e rodapé.
3. `sprint-03.md`: redesign da home.
4. `sprint-04.md`: redesign da página índice de conceitos.
5. `sprint-05.md`: redesign da estrutura compartilhada das páginas de conceito.
6. `sprint-06.md`: redesign da página de exercícios.
7. `sprint-07.md`: responsividade, acessibilidade e motion.
8. `sprint-08.md`: revisão final de consistência visual e textual.
9. `sprint-09.md`: reforço de contraste com tema mais escuro.

## Regra geral

Cada sprint deve:

- revisar os testes existentes antes de alterar o código;
- criar ou ajustar os testes primeiro;
- executar os testes e confirmar a falha esperada;
- implementar o menor incremento visual útil;
- executar novamente os testes relevantes;
- registrar a entrega da sprint em linguagem objetiva.

## Regra visual

Esta task autoriza mudança visual ampla, mas deve preservar:

- tema dark;
- presença da fonte pixelada;
- legibilidade do conteúdo;
- coerência entre páginas;
- experiência compatível com desktop e mobile.

## Regra de linguagem

Todo conteúdo visível da `task04` deve usar português brasileiro correto, com acentos, cedilhas e grafia revisada em:

- títulos;
- descrições;
- chamadas de navegação;
- textos de apoio;
- qualquer outro texto renderizado na interface.

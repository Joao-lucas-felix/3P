# AGENTS.md

## Papel desta area
- Esta area concentra as paginas da SPA.
- Cada pagina deve representar um passo claro na jornada de aprendizagem do iniciante.

## Diretrizes de conteudo
- O conteudo deve ser textual, progressivo e pedagogicamente organizado.
- Cada pagina deve ter objetivo claro, explicacao simples e caminho para aprofundamento ou pratica.
- Sempre que houver link para repositorio de resolucao de exercicios, contextualize o que o estudante vai encontrar e como usar esse material.

## Diretrizes de implementacao
- Antes de criar uma nova pagina, avalie o que pode ser reutilizado de layout, secoes, navegacao e blocos de apoio.
- Evite paginas monoliticas quando a composicao em componentes menores tornar a manutencao mais clara.
- Toda nova pagina deve nascer com testes ajustados primeiro.

## Restricoes visuais
- Preserve o padrao visual atual entre as paginas.
- Nao altere cores, fontes ou tema sem pedido explicito.
